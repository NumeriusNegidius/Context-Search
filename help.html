Getting Started with
Context Search Origin

Thank you for using Context Search Origin! If this is the first time using the extension, this short guide is for you.

Create a home for your search engines

Create a folder in your bookmarks named Searches, place it anywhere you want – just make sure it's spelled with a capital S.

Add search engines

This extension uses as much native features as possible. As such, the search engines are maintained by Firefox as smart keyword bookmarks.

  1. Go to a page with a search engine.
  2. Right click in the search input field.
  3. In the context menu, click Add a Keyword for this Search...
  4. Choose a keyword and make sure to save it to the Searches folder.
