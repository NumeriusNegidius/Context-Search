Context Search Origin

Благодарим за использование Context Search Origin! Если вы впервые воспользовались этим расширением, то вот краткое руководство.

Создайте дом для своих поисковых систем

Создайте в закладках папку с именем Searches, разместите её где угодно, только убедитесь, что название пишется с заглавной буквы S.

Добавьте поисковые системы

Расширение максимально использует встроенные функции браузера. В нём используются технологии Firefox, известные как краткие имена поиска.

  1. Перейдите на страницу поисковой системы.
  2. Щёлкните правой кнопкой мыши в поле ввода поиска.
  3. В контекстном меню выберите пункт Добавить краткое имя для данного поиска...
  4. Выберите краткое имя и убедитесь, что сохранили закладку в папке Searches.
